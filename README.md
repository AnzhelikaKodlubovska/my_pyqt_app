# 🎬 Cinema Management System (PyQt6)

**Cinema Management System** — це кросплатформенний десктопний застосунок для автоматизації роботи кінотеатру, розроблений мовою Python. Програма дозволяє керувати розкладом сеансів, базою фільмів та обліком глядачів залежно від ролі користувача.

---

## 🛠 Технологічний стек

* **Мова програмування:** `Python 3.x`
* **Графічний інтерфейс:** `PyQt6` (використання сигналів `pyqtSignal`, віджетів `QTabWidget`, `QListWidget`, `QPushButton`).
* **База даних:** Локальне зберігання через модуль `pickle` (формат `.pkl`), що забезпечує серіалізацію об'єктів без втрати структури.
* **Архітектура:** Об'єктно-орієнтоване програмування (ООП) з модульною системою передачі даних між вікнами.

---

## 👥 Ролі користувачів

Застосунок пропонує два сценарії взаємодії, які обираються при запуску:

### 1. Менеджер (Адміністратор)
Має повний доступ до керування даними після введення пароля:
* **Фільми:** Перегляд каталогу, додавання нових стрічок, редагування та видалення.
* **Сеанси:** Керування розкладом через `QInputDialog`.
* **Глядачі:** Перегляд списку осіб, що забронювали квитки, та ручне редагування записів через форму `EditViewer`.
* **Менеджери:** Керування персоналом (перегляд ПІБ та Email, додавання нових співробітників).

### 2. Глядач (Клієнт)
Має спрощений інтерфейс для швидкої взаємодії:
* Ввід особистих даних (Ім'я та Прізвище).
* Перегляд доступної афіші та інформації про фільми.
* Вибір сеансу та придбання квитка через форму вибору ряду та місця.

---

## 📐 Особливості реалізації

* **Система вкладок:** Реалізована через `QTabWidget`, що дозволяє логічно розділити функціонал для різних ролей.
* **Адаптивність:** Використання вертикальних макетів `QVBoxLayout` для автоматичного розташування елементів.
* **Інтерактивність:** Кожна дія (натискання кнопки, вибір елемента списку) обробляється через механізм сигналів та слотів (`.connect`).
* **Кросплатформенність:** Завдяки PyQt6 програма працює на Windows, macOS та Linux.

---

## 🚀 Встановлення та запуск

### 1. Попередні вимоги
Переконайтеся, що у вас встановлено Python версії 3.8 або вище.

### 2. Встановлення бібліотек
Встановіть PyQt6 за допомогою менеджера пакетів pip:
```bash
pip install PyQt6
```

### 3. Запуск програми
Виконайте команду в терміналі:
```bash
python main.py
```
